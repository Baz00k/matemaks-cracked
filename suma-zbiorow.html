Suma zbiorów

Sumę zbiorów \(A\) i \(B\) oznaczamy: \[A\cup B\] Graficzna ilustracja sumy zbiorów \(A\) i \(B\):
Do sumy \(A\cup B\) zaliczamy wszystkie liczby, które wchodzą w skład zbioru \(A\) lub zbioru \(B\).
Jeżeli \(A = \{1, 2, 3\}\) oraz \(B = \{5, 6, 7, 8\}\), to: \[A \cup B = \{1, 2, 3, 5, 6, 7, 8\}\]
Jeżeli \(A = \{-2,1,5\}\) oraz \(B = \{1,5,7,12\}\), to: \[A \cup B = \{-2,1,5,7,12\}\]
